 # JSP로 만드는 Hello 서블릿.

설명.
jsp파일명은 URL에 매핑되기 때문에 소문자로 쓰는 것이 좋다.

반복문사용.
<% for( int i=0; i<10; i++ ) { %>
	반복할문장
<% } %>

주의사항.
jsp파일을 만들면 <%@ %> 지시블록을 통해 알아서 charset등이 설정되어서 따로 코딩하지 않는다.
request,response를 꼭 request, response로 사용해야된다.

주의사항.
코드블럭을 아무대나 만들어서 사용하지 말고, 한 곳에 몰아서 사용해서 가독성이 좋게 만드는 것이 좋다.
